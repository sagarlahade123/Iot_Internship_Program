Problem Statement:
First create JSON file and Excel file as input.Excel sheet contains sensors information.
JSON file contains key as sensor name and three parameter as value i.e 0 or 1,X,M. 
If in JSON file first parameter is 0, then not perform any changes on excel column values add
same column values to new excel file.
If in JSON file first parameter is 1, then perform Linear Correlation on excel column values add
new column values to new excel file.
